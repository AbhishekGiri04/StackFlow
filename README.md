## рЯОУ Learning Outcomes

After exploring this project, you will understand:

вЬЕ **HTML Structure** вАФ Semantic markup, tables, forms, image maps, HTML5 media tags  
вЬЕ **CSS Styling** вАФ Box model, shadows, backgrounds, transitions, positioning, pseudo-elements  
вЬЕ **JavaScript DOM** вАФ Selection, manipulation, and dynamic rendering  
вЬЕ **JS Fundamentals** вАФ Variables, data types, type conversion, scope (`var`/`let`/`const`)  
вЬЕ **Form Validation** вАФ Regex-based input validation for name, email, password, credit cards  
вЬЕ **CSS Animations** вАФ Transitions, transforms, pseudo-classes  
вЬЕ **Date API** вАФ Working with `Date` object and `getDay()` for time-based logic  
вЬЕ **Live Updates** вАФ `setInterval` for real-time clocks and self-modifying pages  
вЬЕ **String Methods** вАФ `split()`, `reverse()`, `join()`, `toUpperCase()`, `toLowerCase()`  
вЬЕ **Multi-column Layouts** вАФ `column-count`, `column-gap`, `column-rule`  
вЬЕ **CSS Positioning** вАФ `static`, `relative`, `absolute`, `fixed`, `sticky`
